# Exoplanet Detection using Machine Learning

A supervised machine learning model built to identify exoplanets 
from NASA Kepler telescope light-curve data.

## Project Summary
- **Accuracy achieved:** 98% classification accuracy
- **Tools used:** Python, Scikit-learn, Keras, NumPy, Pandas
- **Model type:** Supervised classification with neural network
- **Data source:** NASA Kepler mission light-curve dataset

## What I Built
Trained a binary classifier to distinguish exoplanet signals 
from false positives in stellar flux data. The model processes 
time-series light curves and identifies characteristic dips 
caused by planetary transits.

## What I Learned
- End-to-end ML pipeline: data preprocessing → feature 
  engineering → model training → evaluation
- Convolutional neural network fundamentals using Keras
- AI ethics and responsible deployment of ML systems
